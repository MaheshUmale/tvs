# tvs
create screener like https://in.tradingview.com/screener
you can use backend https://shner-elmo.github.io/TradingView-Screener/3.0.0/tradingview_screener.html , https://github.com/shner-elmo/TradingView-Screener
dynamiclly adding filter fields and adding result columns.

 FIELDS which can be filterd and shown as columns results are as https://shner-elmo.github.io/TradingView-Screener/fields/stocks.html
 FIELDS have Name ( used in Query and where2 clause ), Display name and Type
 FIELDS have default Timeframe as DAY, if field have <ul><li> then it has multiple timeframe support and available timeframes are as <field>|<timeframe> e.g. change|5 means field change for 5 min timeframe
