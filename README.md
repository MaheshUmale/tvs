# TradingView Screener

A simple stock screener application that uses a Python backend to fetch data from TradingView and displays it in a clean, static HTML interface.

## Setup

1.  Install the required Python packages:
    ```bash
    pip install -r requirements.txt
    ```

## Running the Application

1.  Start the Flask server:
    ```bash
    python app.py
    ```
2.  Open your web browser and navigate to `http://localhost:5002`.
